A smart mobile application for managing finance, time, tasks, habits, goals, and automatically creating schedules from timetable images.

# DayWiggle

DayWiggle is a smart finance and time-management mobile application.

It helps users manage:

- Income and expenses
- Monthly budgets
- Daily tasks
- Work and study time
- Habits
- Personal goals
- Weekly schedules
- Reminders and notifications

DayWiggle also includes a Smart Schedule Scanner. Users can upload an image of a timetable, work schedule, booking note, or handwritten plan. The system reads the image and automatically creates schedule events and reminders.
